# 📋 ToDo List App  
---

## ✨ About

A simple, clean, and responsive **ToDo List app** built with **HTML, CSS, and JavaScript**.  

- Add tasks quickly  
- Mark tasks as done ✔️  
- Delete tasks individually 🗑️  
- Clear all tasks at once  

Perfect for keeping your daily tasks organized!
